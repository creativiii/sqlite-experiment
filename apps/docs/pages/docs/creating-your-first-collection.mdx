# Collections

A collection in Meow is an assortment of content that all shares the same shape.

- All `posts` have both a title and an author
- All `authors` have an email and an email address
- And so on...

## Add a collection

Collections are added to cms' settings like everything else, you can define collections like this:

```tsx
import dynamic from 'next/dynamic'
 
const NextCMS = dynamic(() => import('meow-cms'), {
  ssr: false,
})
 
const CMS = () => (
  <NextCMS
    {...{
      // your collections go in here!
      collections: []
    }}
  />
)
 
export default CMS
```

Let's add the most bare-bones configuration for adding new pages. Your config will look something like this:

```tsx
import dynamic from 'next/dynamic'
 
const NextCMS = dynamic(() => import('meow-cms'), {
  ssr: false,
})
 
const CMS = () => (
  <NextCMS
    {...{
      collections: [
        {
          name: 'page',
          label: 'Page',
          folder: 'content/pages',
          create: true,
          slug: '{{name}}',
          extension: 'mdx',
          fields: [
            { label: 'Title', name: 'title', widget: 'string' },
            { label: 'Content', name: 'body', widget: 'markdown' },
          ],
        }
      ]
    }}
  />
)
 
export default CMS
```
