# Customising previews

One of the best features of Meow is the ability of viewing how the text in your page is going to look as you write it, including React components! Meow handles all of the MDX and compilation logic so you don't need to do it yourself.

Let's start from a basic configuration:

```tsx
const CMS = () => (
  <NextCMS
    {...{
      settings: {
        // more settings here !
        collections: [
          {
            name: 'docs',
            label: 'Docs',
            folder: 'apps/docs/pages/docs',
            create: true,
            slug: '{{name}}',
            extension: 'mdx',
            fields: [
              { label: 'Name', name: 'name', widget: 'string' },
              { label: 'Content', name: 'body', widget: 'markdown' },
            ],
            // the preview object is what we'll use to configure 
            // our CMS preview
            preview: {}
          },
        ],
      },
    }}
  />
)
```

# HTML Preview

The easiest way to start is to add an empty `preview.components` object:

```js
{
  preview: {
    components: {},
  },
}
```

This will tell Meow to convert your Markdown into HTML, even though it will be unstyled. To add styles you can use another `preview` configuration: `wrapper`. Wrapper renders an React component around your preview! You can use it to render a CSS-in-JS component that targets your preview or for a utility class like `tailwind-prose`.

```js
{
  preview: {
    components: {},
    wrapper: ({ children }) => <div className="prose">{children}</div>
  },
}
```

# React Preview

To this you'll want to add a `preview` object. Within `preview` you can set your `components` object, this is a key-value configuration for each of the React components you'll want to use

```tsx
const CMS = () => (
  <NextCMS
    {...{
      settings: {
        collections: [
          {
            name: 'docs',
            label: 'Docs',
            folder: 'apps/docs/pages/docs',
            create: true,
            slug: '{{name}}',
            extension: 'mdx',
            fields: [
              { label: 'Name', name: 'name', widget: 'string' },
              { label: 'Content', name: 'body', widget: 'markdown' },
            ],
            preview: {
              components: Components,
            },
            blocks,
          },
        ],
      },
    }}
  />
)
```
